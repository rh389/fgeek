== Description ==

fGeek is a 100% Responsive WordPress Theme, a perfect choice for a geek website or blog about
programming, learning, geek stuff, news, etc. Features: Social icons in header: Facebook,
Google+, Twitter, LinkedIn, Instagram, RSS Feeds, Tumblr, YouTube, Pinterest, Flickr, Vine,
Multi-Level Drop-Down Header Menu, Admin Customize Options (Social Sites, Slider Images and
Content, Header Logo and Footer Copyright Text), Multiple Browser Compatible (IE9, IE10, IE11,
Chrome, Firefox, Opera, and Safari), Full-Width Page Template, FontAwesome Integration, Sidebar
Widget Area, W3C Markup Validated, Search Engine Optimized (SEO), Mobile-Friendly, and Translation Ready.

== Installation ==

Manual installation:

1. Upload the theme folder to the /wp-content/themes/ directory

Installation using 'Add New Theme'

1. From your Admin UI (Dashboard), use the menu to select Themes -> Add New
2. Search for theme name
3. Click the 'Install' button to open the theme's repository listing
4. Click the 'Install' button

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme specific options
